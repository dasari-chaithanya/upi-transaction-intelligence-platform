# Dashboard Features

The Power BI dashboard provides a highly interactive experience for exploring UPI transactions. 

## Transaction Overview
This page provides a macro view of transaction behaviors:
- Monthly transaction trends.
- Analysis of transaction types (e.g., Peer-to-Peer, Peer-to-Merchant).
- Interactive slicers for bank names (Sent/Received), City, and Merchant.

## Balance Analysis
This page focuses on the financial health and distribution of funds:
- Remaining balance trends over time.
- Analysis by currency.
- Segmentation by customer demographics (Age Groups, Gender).

## Global and Interacting Slicers
Users can filter the entire analysis using rich dimensional slicers, including:
- **Bank Name Sent / Received**
- **City**
- **Device Type**
- **Gender**
- **Age Groups** (A1: <=25, A2: 26-35, A3: >35)
- **Merchant Name**
- **Payment Method / Mode**
- **Purpose**
- **Transaction Type**
- **Status**

## Filtering Behavior
The dashboard utilizes Power BI's standard cross-filtering and cross-highlighting features. Selecting a specific bank, city, or demographic on one chart will dynamically filter the rest of the visuals on the page, allowing for deep-dive analysis from broad metrics down to specific segments.
