# Dashboard Screenshots

This folder is designated for storing screenshots of the Power BI dashboard for the repository documentation.

To properly showcase the dashboard on GitHub, please export or capture the following two screenshots directly from Power BI Desktop and place them in this folder:

1. **Transaction Overview**: Capture the entire page layout of the 'Transaction Overview' report page and save it as `transaction-overview.png`.
2. **Balance Analysis**: Capture the entire page layout of the 'Balance Analysis' report page and save it as `balance-analysis.png`.

Ensure the screenshots are clean (avoiding OS taskbars or unrelated windows) to maintain a professional, recruiter-ready presentation.

Once the screenshots are added, the repository's root `README.md` will automatically display them in the 'Dashboard Preview' section.
